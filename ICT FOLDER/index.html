Lesson 1: Style Sheets

What is css?

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files.
  • CSS is the language we use to style a Web page.
  • CSS separates content from design, making websites easier to maintain and more attractive.
Layers of a webpage:
  • Behavior Layer
  • Presentation Layer
  • Content Layer
Content layer- where you define and layout the text, images, animation, sound, video and everything you want to put in a web page with the use of tags. presentation layer-The presentation layer is where you can define how people see your web page in a browser.
behavior layer- The behavior layer is where you have a real-time user interaction with the web page. The interaction may be simple validation of input or filling out a form or even as big as web-based programs.

Lesson 2: Structure of Style Sheets

Structure of CSS:
Selector – the HTML element you want to style; is a pattern used to "find" or "select" the HTML elements on a webpage to which a set of CSS rules will be applied.
Property – the style attribute you want to modify; is a named characteristic or attribute that defines a specific aspect of how an HTML element should be displayed or rendered by the browser. Properties are used within CSS declarations to apply styles to selected elements.
Value – the weight of the attribute you want to apply to theelement property; is the specific setting or configuration assigned to a CSS property. A property and its value form a CSS declaration, which dictates how an element should be styled
Declaration – is a single instruction that defines a specific style for an element
Declaration block – fundamental part of a CSS rule. It is a block of code that contains one or more CSS declarations, which define the styling rules for the selected HTML element(s).
Three Kinds of CSS:
External - easily allow you to change the layout of your entire web site by simply changing the external style sheet and every page is instantly updated
Internal - used when you have a page that you want to present in a different style from other pages
inline - used for isolated changes to a headline, paragraph, picture or other element
Syntax of the three kinds of CSS:
external- link rel=“stylesheet” href=“style.css” type=“text/css”
internal- style type=“text/css”>
inline- p style=“background-color:violet;”

Lesson 5: CSS Selectors

CSS Selectors
-CSS selectors are used to “find” or “select” the HTML elements that you want to style.
They are divided into 6 types:

  • Class selector
  • Element selector or Tag selector
  • ID selector
  • Universal selector
  • Group selector
  • Attribute selector
Element Selector
-selects HTML elements (p, div, h1, etc.) and applies CSS to them.
- Use an element selector to style all occurrences of a specific HTML tag like all p or h1
ID Selector -ID selector selects the HTML element with a unique identifier (id) and adds CSS to it.
-Use an ID selector when styling a unique element that appears only once on a page (like a header, footer, or main section).
Universal Selector -The universal selector selects every single HTMLelement on the page. It is written using the asterisk ( * ) character
-Use the universal selector when you want to style all elements on the page — for example, to reset margins, padding, or box-sizing
Group Selector -The group selector allows you to select multiple elements and apply the same style to all of them.
-Use a group selector when different elements share the same styling — to avoid repeating code.
Attribute selector
-The attribute selector selects elements based on specific attribute values.The syntax for the attribute selector is:Element[attribute]
-Use an attribute selector when you want to style elements based on their HTML attributes or attribute
